# yasb-settings
My YASB settings

Place ```config.yaml``` and ```styles.css``` in ```C:\Users\<USERNAME>\.config\yasb.```

Make sure to update the environment variables in ```C:\Users\<USERNAME>\.config\yasb\.env ``` (create ```.env ``` file if it doesn't exist)
